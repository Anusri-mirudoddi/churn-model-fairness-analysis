# Model Explainability, Fairness & Bias Analysis

## 📌 Objective
The goal of this project is to analyze the interpretability and fairness of a machine learning model using explainability techniques and bias detection methods.

## ⚙️ Environment Setup

Install required libraries:

pip install pandas numpy scikit-learn matplotlib shap

---

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells step by step
3. View SHAP plots and bias analysis outputs
